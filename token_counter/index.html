Token Counter

Paste text to see how many tokens it produces under different LLM tokenizers. GPT-4o/5 counts are exact (BPE); Claude is an estimate.

Input Text
Token Counts

Why do counts differ? Each model uses a different Byte-Pair-Encoding (BPE) vocabulary. GPT-4o/4.1/5 use o200k_base (~200k tokens), and Claude uses Anthropic's own tokenizer (not publicly released) — so its count is approximated. Counts marked Exact are computed in your browser via the official BPE tables; Approx counts are heuristic estimates.
