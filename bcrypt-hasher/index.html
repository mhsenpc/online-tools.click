ℹ️ About Bcrypt

  • Bcrypt is a password hashing function designed to be slow and computationally expensive, making brute-force attacks difficult.
  • Cost factor determines the number of rounds (2^n). Each increment doubles the computation time.
  • Salting is built-in — each hash includes a unique salt to prevent rainbow table attacks.
  • This tool runs 100% in your browser using the bcrypt-js library. No data is sent to any server.

⚡ Performance Note

Hashing is performed using a Web Worker to avoid blocking the UI. Higher cost factors will take longer — a cost of 12 may take several seconds on modern hardware.
