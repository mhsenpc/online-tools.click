# Password Generator

A simple, secure tool to generate random passwords with customizable parameters.

## Features
- **Configurable Length** — slider or input for length (8-64 characters).
- **Character Sets** — toggle for Uppercase, Lowercase, Numbers, and Special Characters.
- **One-click Copy** — easily copy the generated password.
- **Privacy** — 100% client-side generation.

## How to use
Open `index.html` in your browser.
