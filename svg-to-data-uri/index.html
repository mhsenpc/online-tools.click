💡 Usage Tips

  • Base64 encoding: Produces data:image/svg+xml;base64,... — most compatible with CSS and HTML.
  • URL-encoded: Produces data:image/svg+xml;charset=UTF-8,... — human-readable but longer for complex SVGs.
  • Minification: Removes unnecessary whitespace and comments for smaller file sizes.
  • Use the CSS snippet directly in your stylesheets or inline styles.
